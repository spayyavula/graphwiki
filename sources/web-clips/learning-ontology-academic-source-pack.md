# Learning Ontology Academic Source Pack

This source pack collects verified academic and university-hosted material for building out the Learning Ontology topic in GraphWiki. The emphasis is on ontology engineering, semantic web foundations, linked data, and practical ontology-building workflows.

## MIT

- **20.453J / 2.771J / HST.958J Biomedical Information Technology — Ontology Tutorial**
  - Institution: MIT OpenCourseWare
  - URL: https://ocw.mit.edu/courses/20-453j-biomedical-information-technology-fall-2008/d76d8bbee892ff47b042a0814c10e4f6_hrige_ontlgy_tut.pdf
  - Type: Tutorial / lecture material
  - Why it is useful: A practical, ontology-centered tutorial that explains what ontologies are, how ontology languages differ, and how OWL-style modeling works in applied information systems.

- **Biomedical Information Technology — Readings**
  - Institution: MIT OpenCourseWare
  - URL: https://ocw.mit.edu/courses/20-453j-biomedical-information-technology-fall-2008/pages/readings/
  - Type: Reading list / course page
  - Why it is useful: This page curates foundational readings around OWL, semantic web systems, and linked biomedical data, which makes it useful as a roadmap source rather than a single document.

- **Client Technology**
  - Institution: MIT OpenCourseWare
  - URL: https://ocw.mit.edu/courses/20-453j-biomedical-information-technology-fall-2008/e318cdba2da6eed981ffde19fb3912a7_1021_db_ontology.pdf
  - Type: Lecture PDF
  - Why it is useful: Contains concise explanations of OWL as a web ontology language and of semantic relationships in data-sharing systems.

- **Updating the QMR in 2005: New Approaches**
  - Institution: MIT OpenCourseWare
  - URL: https://ocw.mit.edu/courses/hst-947-medical-artificial-intelligence-spring-2005/d5023f7cc3b14269f89547b12d2089a6_qmr.pdf
  - Type: Course reading / paper
  - Why it is useful: Shows ontology use in applied knowledge-based systems and diagnostic reasoning, useful for grounding ontology engineering in real expert-system practice.

## Columbia

- **Ontology Learning and its Application to Automated Terminology Translation**
  - Institution: Columbia University Computer Science (hosted course copy)
  - URL: http://www.cs.columbia.edu/~vh/courses/LexicalSemantics/Ontologies/navigli-ieee03.pdf
  - Type: Academic paper
  - Why it is useful: A strong source on ontology learning from language, directly relevant to the LLM-wiki and semi-automated ontology-building angle.

- **Designing Clustering Methods for Ontology Building**
  - Institution: Columbia University Computer Science (hosted course copy)
  - URL: http://www.cs.columbia.edu/~vh/courses/LexicalSemantics/Ontologies/Bisson-ecai00.pdf
  - Type: Academic paper
  - Why it is useful: Explains ontology building as a methodological task and gives a more algorithmic perspective on concept discovery and organization.

- **Exploring and Analyzing Linked Data on the Semantic Web**
  - Institution: Columbia University Computer Science (hosted copy)
  - URL: https://www.cs.columbia.edu/~chilton/web/my_publications/BernersLeeTabulator2006.pdf
  - Type: Academic paper
  - Why it is useful: Connects ontology work to linked data exploration and semantic web browsing, which is central to how GraphWiki presents knowledge.

- **Linked Data and Ontologies Seminar 1**
  - Institution: Columbia University EdBlogs
  - URL: https://edblogs.columbia.edu/musiconis/2017/01/20/linked-data-and-ontologies-seminar-1/
  - Type: Seminar summary / teaching material
  - Why it is useful: A readable entry point on linked data and ontologies that can support beginner-facing wiki content.

## Berkeley

- **Ontologies**
  - Institution: UC Berkeley Ptolemy Project
  - URL: https://ptolemy.berkeley.edu/books/Systems/chapters/Ontologies.pdf
  - Type: Book chapter / technical chapter
  - Why it is useful: A very strong conceptual source that defines ontology explicitly as organized knowledge in graph form. Especially useful for GraphWiki's graph-first presentation.

- **Connecting Big Data Semantics**
  - Institution: UC Berkeley School of Information
  - URL: https://www.ischool.berkeley.edu/events/2014/connecting-big-data-semantics
  - Type: Academic event / talk page
  - Why it is useful: Connects semantics and ontology-style thinking to modern data integration challenges, which helps bridge theory and practice.

## Harvard

- **An Open Annotation Ontology for Science on Web 3.0**
  - Institution: Harvard DASH
  - URL: https://dash.harvard.edu/bitstreams/7312037c-8e18-6bd4-e053-0100007fdf3b/download
  - Type: Academic paper
  - Why it is useful: A concrete example of ontology design for scientific annotation, helpful for showing how ontologies structure scholarly knowledge.

- **The SWAN Biomedical Discourse Ontology**
  - Institution: Harvard DASH
  - URL: https://dash.harvard.edu/bitstreams/7312037c-d6b4-6bd4-e053-0100007fdf3b/download
  - Type: Academic paper
  - Why it is useful: A strong applied semantic-web source showing ontology use for scientific discourse, claims, and evidence networks.

- **Supporting Online Teaching Laboratories with Semantic Web**
  - Institution: Harvard-indexed abstract page
  - URL: https://ui.adsabs.harvard.edu/abs/2020rope....4...13G/abstract
  - Type: Abstract / conference paper landing page
  - Why it is useful: Useful for demonstrating ontology-driven resource organization in educational settings.

## Notes

- The strongest foundational sources in this pack are the MIT ontology tutorial, the Berkeley Ontologies chapter, the Columbia ontology-learning papers, and the Harvard ontology-design papers.
- These sources are enough to support a starter wiki layer around ontology basics, semantic web concepts, ontology learning, and applied ontology design.
