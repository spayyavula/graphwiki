# Financial Ontology Industry Source Pack

This source pack extends the Financial Ontology topic with literature and product-facing material from financial data aggregators, infrastructure providers, and investment firms. These are not substitutes for academic sources, but they are valuable for showing how semantic finance and graph-linked identifiers are used in production.

## Notes

- Aggregator and investment-firm literature is strongest when used to explain production use cases, entity-linking infrastructure, and operational knowledge graphs.
- LSEG/PermID is currently the best fit for GraphWiki's financial ontology branch because it explicitly links entities, instruments, and sectors in a graph-like identity layer.
- Hedge-fund-authored ontology literature appears much thinner than aggregator and infrastructure-provider material, so a deeper hedge-fund-specific pass may need a narrower query later.
