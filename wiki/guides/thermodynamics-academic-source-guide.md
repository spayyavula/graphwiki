# Thermodynamics Academic Source Guide

**Definition:** A starter guide to the academic source base for building the Thermodynamics branch of GraphWiki.

## Overview
[coverage: medium]

Thermodynamics is one of the most foundational frameworks in physics, chemistry, engineering, and materials science. It provides the language for describing energy, equilibrium, entropy, heat, work, phase behavior, and spontaneous change. For GraphWiki, a good thermodynamics source base needs to support both introductory learning and specialized extensions into chemical thermodynamics, materials, combustion, and information-processing systems.

This guide highlights a first set of university-hosted sources centered on MIT, Berkeley, Princeton, and Caltech. Together they give a strong progression from basic thermodynamic principles to chemistry-facing equilibrium reasoning and then to advanced applications.

## Recommended Reading Order
[coverage: medium]

1. Start with MIT's Thermodynamics & Kinetics and Berkeley ME 40 for fundamentals.
2. Use those core sources to build the chemical-thermodynamics layer: [[Gibbs Free Energy]], [[Equilibrium Constant|equilibrium]], reacting systems, and [[Chemical Potential]] in multicomponent reasoning.
3. Read the MIT thermodynamics-of-materials lecture and Princeton combustion thermodynamics notes to connect equilibrium reasoning with real chemical and materials systems.
4. Use the Princeton and Caltech advanced sources to connect thermodynamics with dynamical systems, computation, and broader theoretical extensions.

## Source Clusters
[coverage: medium]

- **Foundations:** MIT 5.60, Berkeley ME 40
- **Chemical thermodynamics:** [[Gibbs Free Energy]], [[Chemical Potential]], [[Equilibrium Constant|equilibrium]], reacting systems, mixture behavior, combustion equilibrium
- **Advanced thermodynamics:** MIT 2.43, Princeton dynamical systems chapter
- **Applied thermodynamics:** Princeton combustion notes, Caltech thermodynamic computing paper
- **Cross-topic bridge:** MIT and Caltech materials-oriented thermodynamics sources
